# Nostalgia POST-IT

A simple, responsive web-based Post-It board and To-Do list application built with HTML, CSS, JavaScript, and jQuery.

## Features

### Post-It Board
- Create multiple columns to organize your ideas.
- Add editable Post-It cards to each column.
- Delete individual cards when no longer needed.
- Responsive design using Flexbox - columns stack on smaller screens.

### To-Do List
- Add a dedicated To-Do list section alongside the board.
- Create tasks with checkboxes to mark completion.
- Edit task text directly.
- Delete tasks as needed.

### Theme Toggle
- Light and dark mode Switch.
- Simple smooth transition.

### Persistence
- Automatically saves your board and to-do list to browser localStorage.
- Data persists across browser sessions.

## Technologies Used

- **HTML5** - **CSS3** - **JavaScript** - **jQuery**

## Installation

1. Clone or download the repository.
2. Open `home.html` in a web browser.
